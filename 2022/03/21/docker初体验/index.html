需要在项目下拥有一个Dockerfile文件

下载一个 node12.18.1 环境

WORKDIR /app 设置当前路径为/app

复制 package.json”, “package-lock.json*“, 到/app 下

运行 npm install –production

COPY . . 复制所有当前目录下的问到镜像中

执行 node server.js
